Get set up

Pick your OS, then paste the three commands. The script installs Claude Code (if missing), runs gcloud auth application-default login, and writes your ~/.claude/settings.json.

  1. Install the gcloud CLI if you don't have it: cloud.google.com/sdk/docs/install.
  2. Install Node.js (Claude Code needs it): brew install node
  3. Run the developer-setup script: 
    curl -fsSL __LLM_GATEWAY_URL__/../developer-setup.sh | bash

    Prefer not to pipe curl to bash? Download developer-setup.sh, inspect it, and run it.

  1. Install the gcloud CLI for your distro: cloud.google.com/sdk/docs/install.
  2. Install Node.js via your package manager (e.g. sudo apt install nodejs npm on Debian/Ubuntu).
  3. Run the developer-setup script: 
    curl -fsSL __LLM_GATEWAY_URL__/../developer-setup.sh | bash

    Or download developer-setup.sh and review before executing.

Use WSL2 (Ubuntu) for the best experience — Claude Code runs cleanly under WSL and mirrors the Linux instructions. PowerShell-only setups are not currently officially supported by Claude Code.

  1. Install WSL2 + Ubuntu: wsl --install -d Ubuntu
  2. Open Ubuntu.
  3. Follow the Linux steps above inside the WSL shell.
